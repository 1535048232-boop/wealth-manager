# 家庭资产看板 v1.0.1 产品设计文档

**版本**: v1.0.1  
**更新日期**: 2026年2月15日  
**产品定位**: 极简主义风格的 Fintech 应用，帮助用户清晰掌握家庭财务状况

---

## 一、产品概述

### 1.1 产品简介

家庭资产看板是一款专注于家庭财务管理的移动应用，通过直观的可视化界面，帮助用户：
- 实时掌握净资产、总资产、总负债
- 快速完成账户余额核对（一键对账）
- 查看资产变化趋势和统计分析

### 1.2 目标用户

- 注重财务管理的家庭用户
- 个人投资者
- 需要定期盘点资产的用户

### 1.3 设计理念

参考 Apple Wallet 和 Wealthfront 的设计风格，追求：
- **极简主义**: 界面简洁，信息层次清晰
- **高端质感**: 深午夜蓝主色调，卡片式设计
- **高效交互**: 5分钟完成资产盘点

## 二、功能模块

### 2.1 首页总览 (Dashboard)

| 功能 | 描述 |
|------|------|
| 净资产卡片 | 深蓝色背景，超大字体显示净资产，下方展示总资产/总负债 |
| 资产分布图 | SVG 环形饼图，展示流动资产、投资资产、固定资产占比 |
| 分类明细 | 四大分类卡片，点击展开查看账户明细和占比 |

**视觉设计**:
- 净资产卡片: 深午夜蓝背景 (#1A1F36)，白色文字
- 资产分布图: 环形图，中心显示总资产
- 分类卡片: 白色背景，圆角 16px，轻微阴影

### 2.2 统计分析 (Statistics)

| 功能 | 描述 |
|------|------|
| 趋势选择 | 净资产/总资产/总负债三个维度切换 |
| 趋势图表 | SVG 折线图，展示近6个月变化趋势 |
| 分类趋势 | 点击分类卡片查看各分类的趋势变化 |
| 月度数据 | 列表展示最近6个月的详细数据 |
| 更新时间 | 显示数据最后更新时间 |

**交互设计**:
- Tab 切换不同趋势维度
- 点击分类卡片切换到分类趋势视图
- 滚动查看历史月度数据

### 2.3 一键对账 (Reconcile)

| 功能 | 描述 |
|------|------|
| 账户列表 | 按资产/负债分组展示所有账户 |
| 快速核对 | TextInput 输入框，支持键盘"下一项"跳转 |
| 自动保存 | onBlur/onEndEditing 自动更新数据 |
| 变化提示 | 显示上期余额和变化金额 |
| 视觉反馈 | 金额增加绿色，减少红色（负债逻辑反转） |

**交互优化**:
- 键盘"下一项"自动跳转到下一个输入框
- 输入框自动全选，方便快速修改
- 滑动关闭键盘

## 三、数据模型

### 3.1 账户 (Account)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 账户唯一标识 |
| name | string | 账户名称 |
| balance | number | 当前余额 |
| previousBalance | number | 上期余额 |
| type | 'asset' \| 'liability' | 账户类型 |
| category | AssetCategory | 资产分类 |
| icon | string | 图标 emoji |

### 3.2 资产分类 (AssetCategory)

| 分类 | 标签 | 颜色 | 示例账户 |
|------|------|------|---------|
| liquid | 流动资产 | #4ADE80 (绿色) | 银行存款、支付宝、微信钱包、现金 |
| investment | 投资资产 | #60A5FA (蓝色) | 股票、基金、债券、期权 |
| fixed | 固定资产 | #A78BFA (紫色) | 房产、车辆 |
| liability | 负债 | #F87171 (红色) | 房贷、车贷、信用卡 |

### 3.3 快照 (Snapshot)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 快照唯一标识 |
| date | string | 快照时间 (ISO 8601) |
| accounts | AccountBalance[] | 各账户余额 |
| netWorth | number | 净资产 |
| totalAssets | number | 总资产 |
| totalLiabilities | number | 总负债 |

### 3.4 月度数据 (MonthlyData)

| 字段 | 类型 | 说明 |
|------|------|------|
| month | string | 月份标签 |
| year | number | 年份 |
| monthNum | number | 月份数字 (1-12) |
| netWorth | number | 净资产 |
| totalAssets | number | 总资产 |
| totalLiabilities | number | 总负债 |
| categories | CategoryTotal[] | 各分类总额 |
| accounts | AccountBalance[] | 各账户余额 |

## 四、设计规范

### 4.1 配色方案

| 名称 | 色值 | 用途 |
|------|------|------|
| Primary | #1A1F36 | 主色调，标题、金额 |
| Secondary | #6B7280 | 次要信息、辅助文字 |
| Background | #FAFAFA | 页面背景 |
| Card | #FFFFFF | 卡片背景 |
| Success | #4ADE80 | 增加、正向 |
| Danger | #F87171 | 减少、负债 |
| Muted | #9CA3AF | 弱化信息 |
| Border | #E5E7EB | 分割线、边框 |

### 4.2 字体规范

| 用途 | 字号 | 字重 |
|------|------|------|
| 净资产金额 | 36px | 700 (Bold) |
| 页面标题 | 20px | 700 (Bold) |
| 分类金额 | 18px | 700 (Bold) |
| 区块标题 | 16-18px | 600 (SemiBold) |
| 正文 | 14-16px | 400-500 |
| 辅助文字 | 11-12px | 400-500 |

### 4.3 组件规范

| 组件 | 规格 |
|------|------|
| 卡片圆角 | 16px |
| 输入框圆角 | 12px |
| 按钮圆角 | 16px |
| 卡片阴影 | offset(0,2), opacity(0.05), radius(8) |
| 底部导航高度 | 80px (含安全区) |

## 五、用户流程

### 5.1 查看资产总览

```
打开应用 → 首页总览
    ↓
查看净资产卡片
    ↓
查看资产分布饼图
    ↓
点击分类卡片 → 展开查看账户明细和占比
```

### 5.2 快速对账

```
点击"对账"标签 → 对账页面
    ↓
查看账户列表，显示上期余额和变化
    ↓
点击输入框修改余额
    ↓
点击键盘"下一项"跳转到下一账户
    ↓
失去焦点自动保存
    ↓
切换到首页查看更新后的数据
```

### 5.3 查看趋势统计

```
点击"统计"标签 → 统计页面
    ↓
查看净资产/总资产/总负债趋势图
    ↓
点击 Tab 切换趋势维度
    ↓
点击分类卡片查看分类趋势
    ↓
滚动查看月度数据列表
```

## 六、技术架构

### 6.1 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| React Native | 0.81.5 | 跨平台框架 |
| Expo | 54.0.33 | 开发工具链 |
| TypeScript | 5.9.2 | 类型安全 |
| React Native SVG | 15.15.3 | 图表绘制 |

### 6.2 状态管理

使用 React Context API 管理全局状态：
- `AssetContext`: 账户数据、快照、统计计算

### 6.3 项目结构

```
src/
├── App.tsx                 # 主应用入口
├── context/
│   └── AssetContext.tsx    # 全局状态管理
├── types/
│   └── index.ts            # TypeScript 类型定义
├── utils/
│   └── format.ts           # 金额格式化工具
├── styles/
│   └── theme.ts            # 主题配置
├── components/
│   ├── NetWorthCard.tsx    # 净资产卡片
│   ├── AssetChart.tsx      # 资产分布饼图
│   ├── AssetCategoryCard.tsx # 分类资产卡片
│   └── TrendChart.tsx      # 趋势图表
└── screens/
    ├── DashboardScreen.tsx # 首页总览
    ├── StatisticsScreen.tsx # 统计分析
    └── ReconcileScreen.tsx # 一键对账
```
