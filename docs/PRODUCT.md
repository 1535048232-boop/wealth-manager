# 家庭资产看板 - 产品文档

## 1. 产品概述

**产品名称**: 家庭资产看板

**产品定位**: 一款极简主义风格的 Fintech 应用，帮助用户清晰掌握家庭财务状况，实现资产可视化和快速对账。

**目标用户**: 注重财务管理的家庭用户、个人投资者

**设计理念**: 参考 Apple Wallet 和 Wealthfront，追求高端、简洁、易用的用户体验

## 2. 核心功能

### 2.1 首页总览

| 功能模块 | 描述 |
|---------|------|
| 净资产卡片 | 超大字体显示净资产总额，下方展示总资产和总负债 |
| 资产分布图 | 环形饼图展示流动资产、投资资产、固定资产的占比 |
| 分类明细 | 四大分类卡片（流动资产、投资资产、固定资产、负债），点击可展开查看账户明细 |

### 2.2 一键对账

| 功能模块 | 描述 |
|---------|------|
| 账户列表 | 按资产/负债分组展示所有账户 |
| 快速核对 | 每个账户右侧显示输入框，方便用户核对实际余额 |
| 差异标记 | 当输入值与账面余额不同时，输入框背景高亮提示 |

## 3. 设计规范

### 3.1 配色方案

| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| Primary | #1A1F36 | 主色调，用于标题、金额等核心信息 |
| Secondary | #6B7280 | 次要信息、辅助文字 |
| Background | #FAFAFA | 页面背景 |
| Card | #FFFFFF | 卡片背景 |
| Success | #4ADE80 | 资产增加、正向金额 |
| Danger | #F87171 | 负债、减少金额 |
| Border | #E5E7EB | 分割线、边框 |

### 3.2 组件风格

- **卡片设计**: 圆角 16px，轻微阴影
- **按钮**: 圆角 16px，主色背景
- **输入框**: 无边框设计，圆角 12px
- **字体**: 大字号金额显示 (36px)，标题 20px，正文 16px

### 3.3 设计参考

- Apple Wallet - 极简卡片设计
- Wealthfront - 高端金融风格
- 极简主义设计原则

## 4. 数据模型

### 4.1 账户 (Account)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 账户唯一标识 |
| name | string | 账户名称 |
| balance | number | 余额 (正数为资产，负数为负债) |
| type | 'asset' \| 'liability' | 账户类型 |
| category | AssetCategory | 资产分类 |
| icon | string | 图标 emoji |

### 4.2 资产分类 (AssetCategory)

| 分类 | 说明 | 示例账户 |
|------|------|---------|
| liquid | 流动资产 | 银行存款、支付宝、微信钱包、现金 |
| investment | 投资资产 | 股票、基金、债券 |
| fixed | 固定资产 | 房产、车辆 |
| liability | 负债 | 房贷、车贷、信用卡 |

## 5. 用户流程

### 5.1 查看资产总览

1. 打开应用，默认进入首页总览
2. 查看净资产总额
3. 查看资产分布饼图
4. 点击分类卡片展开查看账户明细

### 5.2 快速对账

1. 点击底部"对账"标签
2. 浏览账户列表，核对账面余额
3. 如有差异，在输入框中修改
4. 点击"保存对账结果"
